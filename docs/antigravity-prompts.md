## COMPONENT SNIPPETS FOR ANTIGRAVITY

### Wax Seal Button
```
Create a button that looks like a red wax seal stamp.
Circular shape, 150px diameter.
Red wax texture (dark red #8B1A1A with radial gradient highlight).
Ornamental border ring inside.
Text centered, cream color, Special Elite font.
On hover: slight scale up 1.05.
On click: scale down 0.95 then back up with red ink splash animation.
```

### Manila Price Tag
```
Create a hanging price tag component.
Shape: rectangle with notched top-right corner (like a real tag).
Small circle hole at top with thin string/line.
Background: cream/tan #DEB887.
Price in Abril Fatface bold.
Label "Starting from" in small Special Elite.
Slight rotation 2-3 degrees.
Drop shadow.
```

### Rubber Stamp Tech Tag
```
Create tech badge tags that look like rubber stamp impressions.
Rectangle, 60px × 24px, border-radius 3px.
Border: 2px solid, slightly rough/uneven (box-shadow trick).
Text: Special Elite, uppercase, letter-spacing 0.1em.
Ink colors: n8n=blue, Qdrant=purple, RAG=dark green, Supabase=teal.
Slight opacity 0.85 to look like real ink.
```

### Cork Board Background
```
Create a cork board texture background.
Base color: #C4A07A (warm cork brown).
Add subtle fiber texture pattern using CSS background-image SVG noise.
Vignette: radial gradient overlay, transparent center, 
dark brown rgba(60,30,10,0.4) at edges.
Overall feels warm, physical, tactile.
```
